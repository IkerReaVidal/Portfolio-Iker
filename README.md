# 🌐 Mi Portafolio Web — Proyectos y Experimentos

## 📌 Descripción  
Este repositorio contiene mi portafolio personal desarrollado con **HTML, CSS y JavaScript**, donde presento varios proyectos y experimentos técnicos. Incluye también archivos **JSON** para manejo de datos y un modelo **.obj** utilizado en una sección 3D interactiva.

El objetivo del portafolio es mostrar mis habilidades en desarrollo web, diseño visual y manejo de recursos multimedia.


## 🚀 Tecnologías utilizadas  
- **HTML5** — estructura del sitio  
- **CSS3** — estilos, animaciones y diseño responsivo  
- **JavaScript** — interactividad y lógica  
- **JSON** — almacenamiento y carga de datos  
- **OBJ 3D Model** — visualización 3D integrada en el navegador

## 🔧 Cómo ejecutarlo  
1. Clona el repositorio:  
   ```bash
   git clone https://github.com/ikerreavidal/Portfolio-Iker.git
   ```
2. Abre `index.html` en tu navegador.  
3. Disfruta del portafolio.
